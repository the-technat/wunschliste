# Keyboard Shortcuts

## Global

`/` - Search notes. Disabled when focus is on a text input.

Note - View Mode

`e` - Switch edit mode on.

Note - Edit Mode

`CTRL-Enter` / `CMD-Enter`- Save changes.

`ESC` - Exit edit mode.

`CTRL-B` / `CMD-B` - Bold.

`CTRL-I` / `CMD-I` - Italic.

`CTRL-S` - Strike-through.
